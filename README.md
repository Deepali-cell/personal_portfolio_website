# Personal Portfolio

A modern and responsive personal portfolio website built with **Next.js**, **React**, **Tailwind CSS**, and **Framer Motion** to showcase my skills, projects, experience, and achievements as a Software Engineer and AI Developer.

## 🚀 Features

* Modern and responsive UI
* Smooth animations with Framer Motion
* Project showcase with GitHub links
* Skills and technology stack
* Experience and internship timeline
* Resume download
* Contact section
* SEO-friendly architecture
* Optimized performance with Next.js

## 🛠️ Tech Stack

* Next.js
* React.js
* TypeScript
* Tailwind CSS
* Framer Motion
* HTML5
* CSS3

## 📂 Sections

* Home
* About
* Skills
* Experience
* Projects
* Contact
